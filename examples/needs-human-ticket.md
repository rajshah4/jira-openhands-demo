# Needs Human Ticket

Use this for the human-in-the-loop demo path.

## Summary

```text
People are confused by pet prices
```

## Description

```text
This came up in the intake review. Can we fix it?
```

## Expected Agent Behavior

OpenHands should not guess. It should ask a human for clarification because the
ticket could mean several different things:

- search filtering by adoption fee
- fee display or copy
- payment or checkout policy
- counselor workflow/training issue

Preferred response:

```text
I found multiple plausible interpretations and do not have enough evidence to
choose safely. Should I work on search filtering, fee display copy, payment
policy, or something else?
```
